# tfe_outputs

This ephemeral resource can be used to retrieve a workspace's state outputs without saving them in state.

## Example Usage

Basic usage:

```terraform
ephemeral "tfe_outputs" "foo" {
  organization = "my-org"
  workspace    = "my-workspace"
}

provider "vault" {
  auth_login {
    path = "auth/approle/login"

    parameters = {
      role_id   = ephemeral_tfe_outputs.foo.values.vault_role_id
      secret_id = ephemeral_tfe_outputs.foo.values.vault_secret_id
    }
  }
}
```
